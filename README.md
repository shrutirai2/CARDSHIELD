## 🚀 Live Demo

- **Frontend**: [CardShield Frontend](https://cardshield-frontend.onrender.com)
- **Backend API**: [CardShield Backend](https://cardshield-backend.onrender.com)

## ✨ Features

- Real-time fraud detection using Machine Learning
- Single Transaction Analysis
- Batch Processing (CSV Upload)
- Interactive Dashboard with model statistics
- Beautiful modern UI with glassmorphism design
- Dual Mode (Real ML + Mock Mode for demo)

## 🛠 Tech Stack

- **Frontend**: React.js, Tailwind CSS
- **Backend**: Flask (Python)
- **Machine Learning**: XGBoost
- **Deployment**: Render.com (Frontend + Backend)
